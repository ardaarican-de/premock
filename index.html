Figma prototype guide

Check these settings in Figma so your prototype displays correctly on the phone:

  1. Sharing accessOpen Share (top right) → set the link to “Anyone with the link” and “can view”.
  2. No device frameIn prototype settings, set Device → None. PreMock adds the phone frame for you.
  3. ScalingIn prototype Settings → Scaling, choose “Scale down to fit width”.
  4. Starting frameMake sure the correct starting frame is selected.
  5. Hide Figma UIIn Present mode, open Options → enable “Hide Figma UI” (the link will include hide-ui=1).

After updating the settings, use Share → Copy link to copy the link again and paste it here.
